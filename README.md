# TITANIC-survival-prediction
End-to-end Machine Learning project demonstrating data preprocessing, exploratory data analysis, Logistic Regression, and model evaluation on the Titanic dataset.
# Titanic Survival Prediction using Logistic Regression

## Objective
Build a Machine Learning model to predict whether a passenger survived the Titanic disaster.

## Tools Used
- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Project Workflow
- Data Loading
- Data Exploration
- Data Cleaning
- Data Visualization
- Feature Selection
- Train-Test Split
- Logistic Regression Model
- Model Evaluation

## Results
- Accuracy: 81%
